## Installation

```cmd
git clone https://github.com/NoteSwiper/minesteal.git
cd minesteal
npm install
npm run get-skins
```

or

```cmd
git clone https://github.com/NoteSwiper/minesteal.git
cd minesteal
npm install
node .
```
